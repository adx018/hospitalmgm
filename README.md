# Hospital Management System

## Timeline
This project was developed between **September 2025 and November 2025**
as part of academic coursework. The repository was published to GitHub
later for portfolio and documentation purposes.

## Description
A hospital management system designed to manage patients, doctors,
appointments, and administrative workflows efficiently.

## Features
- Patient record management
- Doctor scheduling
- Appointment booking
- Basic authentication and validation
